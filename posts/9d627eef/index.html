Intellij idea 2019.3激活教程

发表于 2020-04-19 | |
字数统计: 276 | 阅读时长 ≈ 1

2019年开始Jetbrains对旗下产品进行了一次升级,之前激活码激活Intellij idea的方式都不能使用了.本文以 Intellij idea 2019.3为例,讲解一下Intellij idea的激活.以下方法对Jetbrains旗下其他产品,如Webstorm,PyCharm等同样适用.

下载破解插件

下载jetbrains-agent.jar破解插件,百度云盘提取码：syfq.下载完成后,将插件放入自己容易找到没有中文字符的文件夹下面(建议放入idea的bin目录下),复制路径.

配置插件

在Idea右上角点击 Help->Edit Custom VM Options.

打开idea64.exe.vmoptions文件,最后一行增加-javaagent:


填写-javaagent:式例

填写激活码

在Idea右上角点击 Help->Register,选择Activation Code


填写激活码

重启Idea
